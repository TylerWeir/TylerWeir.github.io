Technical Projects

Halo 5 Daily Kill Counter

Using a Raspberry Pi computer, I wrote a Python program to download data from the Halo 5 public API and display the total kills performed by a given player on the current day. The display is made of three 4" seven segment displays. This program taught me many valuable skills for programming in the linux environment. I learned to use tools such as ssh and vim which I still use today.

Carry Look Ahead Adder Circuit

For my final project in Computer Architecture, I used breadboards to create a 4 bit carry look ahead adder. This circuit is very similar to the addition hardware found in modern computers. It is much faster than the popular ripple carry adder; however, it requires more complex circuitry.

Flocking Algorithm Program

I read Craig Reynolds' paper Flocks, Herds, and Schools: A Distributed Behavioral Model which introduced a now classic agent based flocking algorithm. I used this algorithm to make a program allowing the user to create small "birds" with the click of the mouse and then watch them flock together. The "boids" or bird objects are represented by small blue triangles that fly around the screen. They perform seemingly complex movement which is driven by a couple basic behaviors.

Verlet Integration Physics Engine

Following Thomas Jakobsen's Paper Advanced Character Physics, I am in the process of creating a physics engine for rigid bodies defined by sticks and points. Rather than using Euler's method of integration, the program uses Verlet integration: a process by which the velocity of an object is calculated from it's previous positions. This process is much more stable than Euler's integration routine when working with large quantities of interacting bodies. Currently, I am working on the challenge of collision detection between rigid bodies.
