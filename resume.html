Using a Raspberry Pi computer, I wrote a Python program to download data from the Halo 5 public API and display the total kills performed by a given player on the current day. The display is made of three 4" seven segment displays. This program taught me alot about programming in the linux environment. I learned to use tools such as ssh and vim which I still use today.

Carry Look Ahead Adder Circuit

For my final project in Computer Architecture, I used breadboards to create a 4 bit carry look ahead adder. This circuit is very similar to the addition hardware in modern computers. It is much faster than the popular ripple carry adder; however, it requires more complex circuitry.

Flocking Algorithm Program

I made adaption of Craig Reynolds classic agent based flocking algorithm in Python. The "boids" or bird objects are represented by small blue triangles that fly around the screen. They perform seemingly complex movement which is driven by three simple behaviors.

Verlet Integration Physics Engine

Following Jakobsen's Paper, I am in the process of creating a physics engine for rigid bodies defined by sticks and points. Rather than using Euler's method of integration, the program uses Verlet integration: a process by which the velocity of an object is calculated from it's previous positions. This process is much more stable than Euler's integration routine, thus making it a better match for systems of many interacting rigid bodies. Currently, I am working on the challenge of collision detection between rigid bodies in a system.
